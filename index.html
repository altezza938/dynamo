Export Files

Generate and download files for use in Civil 3D, Dynamo, and AutoCAD.

COGO Points CSV

Point Number, Easting, Northing, Elevation, Description for each nail head. Import directly into Civil 3D as COGO points.

AutoCAD Script (.scr)

AutoCAD script file that draws 3D lines for each soil nail. Run with the SCRIPT command in Civil 3D.

Dynamo JSON

JSON data file for use with Dynamo for Civil 3D. Contains full nail geometry, parameters, and placement coordinates.

DXF Drawing

DXF file with soil nails drawn as 3D lines on separate layers. Open directly in Civil 3D or AutoCAD.

LandXML

LandXML format for importing terrain and point data into Civil 3D and other civil engineering software.

Summary Report

Printable HTML summary report with project details, nail schedule, quantities, and cross-section drawing.
